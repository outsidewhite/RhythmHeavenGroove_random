# Rhythm Heaven Groove Renshan

リズムゲームをランダムに抽選し、パーフェクトの連続達成数を記録するためのシンプルなWebツールです。

## できること

- ゲームをランダムに抽選できます。
- 1周するまで同じゲームが重複しないように抽選します。
- パーフェクトできたゲームを履歴に残せます。
- 失敗した場合は「やりなおし」で連チャン記録を区切れます。
- 画像をクリックすると、表/裏モードを切り替えられます。
- 進行状況はブラウザに保存されるため、ページを閉じても途中から再開できます。

## 使い方

1. ページを開きます。
2. 「抽選開始」を押してルーレットを開始します。
3. 「ストップ」を押して挑戦するゲームを決めます。
4. パーフェクトできたら「パーフェクト」を押して次の抽選へ進みます。
5. 失敗したら「やりなおし」を押して記録を区切ります。
